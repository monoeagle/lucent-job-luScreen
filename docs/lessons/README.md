# Lessons

Datierte Lern-Sessions (länger, narrativ). Format-Vorlage:

```
yyyy-mm-dd-<kebab-thema>.md
```

## Inhalt

- **Was war das Problem?** (1 Absatz, Kontext)
- **Was haben wir versucht?** (chronologisch, was funktioniert hat und was nicht)
- **Was war die Erkenntnis?** (Kern-Insight)
- **Was leiten wir daraus ab?** (Konvention, Tooling-Anpassung, Doku-Update)

## Abgrenzung

| Format | Ort | Länge | Tonalität |
|---|---|---|---|
| **Lesson** | `docs/lessons/` | 1–3 Seiten | Narrativ, Session-bezogen |
| **Erkenntnis** | `.erkenntnisse/` | 1 Absatz | Kontextfrei lesbar, generelle Regel |
| **Spec/Plan** | `docs/superpowers/{specs,plans}/` | beliebig | Implementierungsvorhaben |
| **Handoff** | `docs/handoffs/` | 1 Seite | Session-Übergabe |
