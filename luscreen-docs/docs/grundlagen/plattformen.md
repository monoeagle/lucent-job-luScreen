# Plattformen

| Komponente | Anforderung |
|---|---|
| Windows | 10 (Build 1809+) oder 11 |
| PowerShell | 7.0+ (`pwsh.exe`) |
| .NET | 6.0+ (kommt mit PS7) |
| Apartment | STA (wird automatisch gestartet) |
| DPI | PER_MONITOR_AWARE_V2 |

Nicht unterstützt: PowerShell 5.1, Windows Server Core (kein WPF), macOS, Linux.
