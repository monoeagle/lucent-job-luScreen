# LucentScreen

> Windows-Screenshot-Tool mit Tray, globalen Hotkeys, Editor und Verlauf.

## Was ist LucentScreen?

Ein leichtgewichtiges Tool, das im Hintergrund läuft und auf Hotkey-Druck einen Screenshot anfertigt — von einem Bereich, einem Fenster, einem Monitor oder allen Monitoren. Mit Annotations-Editor, Verlauf und Zwischenablage-Kopie.

## In 3 Minuten loslegen

1. App über die Softwareverteilung installieren lassen
2. Mit `Ctrl+Shift+1` einen Bereich aufnehmen
3. Im Editor markieren/zuschneiden, dann speichern

→ [Detaillierter Schnellstart](grundlagen/schnellstart.md)
